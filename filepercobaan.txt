file ini adalah file yang dibuat didalam komputer
yang kemudian akan diupload kedalam github
menggunakan perintah:
git add -> untuk menambahkan data atau file
git comit -m -> untuk menambahkan tanda atau komentar
git push -u origin master -> untuk memastikan file masuk ke repo
git config --global user.name -> untuk memasukan settingan username kedalam git kita
git config --global user.email -> --------"-------- settingan email kedalam git kita
git init -> digunkan,  jika ingin komputer dengan github terhubung dan biasanya digunakan ketika melakukan remote
